# 2022.04.15

1. 代码规范性

   1.1 函数的注释格式：

```C++
    /**
     * 描述
     * @author Your name 
     * @param 参数1 （描述）
     * @param 参数2 （描述）
     * @param 参数3 （描述）
     * @return 返回类型 （描述）
     */
	void xxx()
    {
        
    }
```

​        1.2 变量的命名应具有描述性，减少使用单个字母的命名方式。

2. 保存格式

   所有代码文件使用UTF-8无签名的保存格式

   百度搜索“vs设置高级保存选项”，并完成相关设置。
